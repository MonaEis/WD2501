ul vs. ol

  • unordered list
    • HTML-tag: ul
    • listet Punkte mit bullet points auf
    • Beispiele:
      1. To-Do-Liste
      2. Einkaufszettel
      3. Anwesenheitslisten
      4. ...
    • Weiterführende Infos
  • ordered list
    • HTML-tag: ol
    • listet Punkte mit Nummerierung auf
    • Beispiele:
      1. Inhaltsverzeichnis
      2. Abläufe
      3. Ranglisten
      4. ...
    • Weiterführende Infos
